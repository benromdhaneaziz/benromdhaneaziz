## 📌 Featured Projects

### 🏢 Company Project — Private (GitLab)

> **Travel CRM & Analytics Platform — AI Multi-Agent System**
>
> Enterprise-grade Travel CRM built during my internship. Features an **8-agent AI chatbot** (SQL, Analyst, Forecast, Anomaly Detection, Customer Segmentation, Hotel Contract, Flight Search, and an Orchestrator) powered by **LangGraph + LangChain + DeepSeek LLM** (via OpenRouter). The platform exposes **52 REST endpoints** over an **Oracle 11.2g** database, streams responses in real-time via **SSE**, and is served through a full **Angular 17** dashboard with interactive charts, MapLibre flight route maps, hotel contract PDF/DOCX parsing with OCR, and a Streamlit testing UI. Implements JWT auth, rate limiting, LRU cache, and multi-entity data isolation.
>
> `Python` · `FastAPI` · `LangGraph` · `LangChain` · `DeepSeek LLM` · `Angular 17` · `Oracle DB` · `SQLite` · `Docker` · `Amadeus SDK` · `OCR` · `SSE Streaming`
